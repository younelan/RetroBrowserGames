# Galactic Storm

A classic top-down scrolling shooter game.

## Description

In Galactic Storm, you pilot a spaceship, battling against hordes of enemy drones, scouts, and cruisers. Your objective is to survive as long as you can and achieve the highest possible score.

## Features

*   Endless scrolling level
*   Three different types of enemies
*   Upgradeable weapon system
*   Power-ups
*   Score and level tracking

## How to Run the Game

To run the game, you need a web server. You can use any simple web server, like the one that comes with Python.

1.  Make sure you have Python installed on your system.
2.  Open a terminal or command prompt in the root directory of the game.
3.  Run the following command to start a simple web server:

    ```bash
    python -m http.server
    ```

4.  Open your web browser and navigate to `http://localhost:8000`.

Enjoy the game!
